# gx-script

## Clone This Repository

To clone this repository, you can use the following command:

```bash
git clone https://github.com/GioseaxMC/gx-script.git
```

## Documentation

Refer to the .\tests directory for example usage

## Usage

run a .gx program with:

```bash
./build ./script.gx
```
or
```bash
./gxx ./script.gx
```
